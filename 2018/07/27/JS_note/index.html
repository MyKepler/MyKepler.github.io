#JS笔记

1、JavaScript会返回数据类型(typeof)：字符串（String）、数字(Number)、布尔(Boolean)、对象(Object)、未定义（Undefined）、函数（function）。【数组、日期、null之类的返回object】,不用typeof,用A.constructor返回所有 JavaScript 变量,包括下面10个变量类型。

5 种不同的数据类型：string、number、boolean、object、function
3 种对象类型：Object、Date、Array
2 个不包含任何值的数据类型：null、undefined

对象寻址/访问对象属性的两种方法：

name=person.lastname;

name=person["lastname"];

如果对象是一个函数，就用name=person.fullName();如果不加括号，则直接输出函数，而不是结果。

创建对象方法：methodName : function() { ... }
访问对象方法：objectName.methodName()

声明变量类型可以直接用new来声明其类型或者赋值。

2、JavaScript变量生存周期：局部变量在函数运行之后被删除；全局变量在页面关闭后被删除。

对未声明的变量赋值，该变量自动作为window的一个属性。
